# الحياة في اليونان (Life in Greece)

موقع إلكتروني يقدم معلومات شاملة عن الحياة في اليونان، بما في ذلك الخدمات الحكومية، فرص العمل، المنظمات، المكاتب المتخصصة، والسياحة.

## الميزات

- تصميم متجاوب يعمل على جميع الأجهزة (الكمبيوتر، الهاتف، الجهاز اللوحي)
- قائمة تنقل احترافية مع قوائم منسدلة
- قائمة خاصة بالهاتف المحمول في الجزء السفلي من الشاشة
- أقسام متعددة تغطي جميع جوانب الحياة في اليونان:
  - الخدمات (دوائر اللجوء، دوائر حكومية، عقارات وسكن، مستشفيات، خطوط النقل، سياحة)
  - العمل (أفضل مواقع البحث عن عمل)
  - المنظمات (إنسانية، طبية، تعليمية)
  - المكاتب (خدمية، محاسبة، محامين)
  - السياحة (فنادق، صيدليات ومتاجر، مطاعم)
- تأثيرات حركية جذابة وانتقالات سلسة
- تصميم عصري وجذاب

## التقنيات المستخدمة

- HTML5
- CSS3 (مع تخطيط Flexbox و Grid)
- JavaScript (Vanilla JS)
- Font Awesome (للأيقونات)
- خطوط Google (خط Cairo)

## كيفية الاستخدام

1. قم بفتح ملف `index.html` في متصفح الويب.
2. استعرض الأقسام المختلفة باستخدام القائمة العلوية (في وضع سطح المكتب) أو القائمة السفلية (في وضع الهاتف المحمول).
3. انقر على الروابط للوصول إلى المعلومات التفصيلية في كل قسم.
